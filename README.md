# SakuraRedeem
---
一个可以生成与兑换cdk的插件

### REQUIREMENTS
- [x] 编辑者输入package创建可用包裹，提供gui编辑包裹信息
- [ ] 编辑者可以自由添加其他兑换时的附加信息，如执行的指令或者获得的钱币
- [ ] 兑换者输入cdk 可以获得上述编辑者做好的所有内容
- [ ] 兼容placeholder api
- [x] 方便管理，可以接MySQL
- [ ] 可以批量添加以及通过yml导入
- [ ] 随机cdk和自定义cdk两种模式
- [ ] cdk有有效时间限制 （自动清除手动清除）

关于开发内容请查看[DEVELOP.md](./DEVELOP.md)
